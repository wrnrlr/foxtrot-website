Install Foxtrot

To install Foxtrot, the [Go programming language](https://golang.org/doc/install) and [Git](https://git-lfs.github.com/) with the [Large File Storage](https://git-lfs.github.com/) extension are required.

bash
                    # Clone Foxtrot repository and its submodules.
                    git clone --recurse-submodules https://github.com/wrnrlr/foxtrot

                    # Generate builtin rules that come with the expreduce submodule.
                    cd foxtrot/expreduce
                    go generate ./...

                    # Start Foxtrot
                    cd ..
                    go run cmd/main.go
                    ```

A Symbolic Programming Language

Foxtrot has a build-in programming language with a rich library for functions, symbols and other expressions. For more documentation about the Foxtrot langiage read our Get Started Guide and api reference.

Foxtrot is written in the Go language and leverages Go's ecosystem. It is not required to install Go to run Foxtrot, but it can help to manage dependencies, and build Go or C Libraries.
